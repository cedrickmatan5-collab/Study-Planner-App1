Project Title: Study Planner App

Project Description

The Study Planner App is a mobile application developed using React Native that helps students organize and manage their study tasks efficiently. The system allows users to create, view, update, and delete study plans, making it easier to track academic responsibilities and deadlines.

This project aims to improve productivity and time management by providing a simple and user-friendly mobile interface where students can plan their daily or weekly study schedules. The app ensures that users can easily modify their tasks and stay organized anytime using their mobile devices.

Detailed System Flow
1. Input Phase
User enters study task details (subject, description, deadline) through mobile input fields
User submits the task using a button
Data is temporarily stored in the application state or local storage

2. Process Phase
The system processes user input using React Native components and state management
CRUD operations are performed:

Create: Add new study task
Read: Display list of tasks using mobile UI components
Update: Edit existing task
Delete: Remove task
Data is updated dynamically within the app without restarting it

3. Output Phase
Updated list of study tasks is displayed on the mobile screen
Tasks are shown with details and deadlines
Changes are reflected in real-time in the app interface
